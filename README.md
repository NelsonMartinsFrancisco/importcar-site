# ImportCar.pt

Plataforma para venda de carros importados da Bélgica para Portugal.

## Funcionalidades
- Pesquisa por marca, modelo e ano
- Fotos dos veículos
- Detalhes técnicos
- Pedido de financiamento
- Entrega com verificação mecânica
